# TODO MVC

[Demo en vivo](https://utn-mdp.github.io/todomvc/)

### Para iniciarlo:

```bash
# clonen el proyecto:
git clone https://github.com/UTN-MDP/todomvc.git

# entren en la carpeta
cd todomvc

# inicien un servidor web
php -S localhost:8888

# si no tienen PHP instalado, abran el archivo index.html directamente en el navegador.
```
